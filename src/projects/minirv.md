# 迷你 RISC-V 处理器

## 项目简介

使用数字电路工具实现一个能够执行基础 RISC-V 指令的处理器，并通过课程提供的程序测试。

## 核心模块

- PC 与下一条指令选择
- 指令存储器
- 指令译码
- 立即数生成器
- 寄存器堆
- ALU
- 分支比较器
- 数据存储器
- 写回选择

## 当前状态

<span class="status-chip working">调试中</span>

目前主要关注测试结束后 `x10` 不为 0 的问题，排查重点是立即数、分支、访存和写回路径。

## 完成标准

- 电路无位宽冲突与悬空输入；
- 所需指令均能正确执行；
- 使用指定 `mem.hex` 运行；
- 最终 `x10 / a0 = 0`。
