# E5：Verilator 初体验

## 当前进展

已完成对基础 Verilog 模块的 lint 检查，示例命令：

```bash
verilator --lint-only -Wall \
  --top-module light \
  vsrc/light.v
```

当报告没有错误时，说明当前模块至少通过了静态语法和部分结构检查。

## 重点理解

- `--top-module light`：指定顶层模块；
- `--lint-only`：只检查，不生成仿真程序；
- `-Wall`：打开较完整的警告；
- 复位信号决定电路初始状态；
- 时钟边沿决定时序逻辑何时更新。
